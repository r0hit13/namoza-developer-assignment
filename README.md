# Task 2: Consultation Landing Page

A lightweight, mobile-first landing page was built using only:

- HTML
- CSS
- Vanilla JavaScript

Features:

- Strong conversion-focused headline and CTA
- Minimal two-field form (Name and Phone)
- Trust indicators (ratings, clinics, patient count)
- Responsive mobile experience
- GTM `window.dataLayer.push()` integration
- Thank-you state without page reload
- Hidden clinic preference field for CRM integration
- Optimized for 90+ PageSpeed Insights mobile score

Files:

```text
task2-landing-page/index.html
task2-landing-page/pagespeed-mobile.png
```

## Running the Landing Page

Simply open `index.html` in any modern browser.

No server, framework, or build tools are required.
